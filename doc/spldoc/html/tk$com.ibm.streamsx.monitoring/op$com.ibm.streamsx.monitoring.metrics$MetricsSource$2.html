IBM Streams authentication

For IBM Streams authentication the following authentication parameters should be used:

  • user

  • password

Authentication can be configured with operator parameters or application configuration.

Authentication with application configuration

Save Credentials in Application Configuration Property

The following steps outline how this can be done:

  1. Create an application configuration called monitoring. You need to set the operator parameter applicationConfigurationName.
  2. Create two properties in the monitoring application configuration named user and password.
  3. The operator will look for an application configuration, if the parameter applicationConfigurationName is set and will extract the information needed to connect.
