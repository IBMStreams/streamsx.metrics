Operators

  • MetricsMonitor: The MetricsMonitor operator connects to the MetricsSource operator and receives incoming metric tuples.
  • MetricsSource: The MetricsSource operator uses the JMX API to retrieve metrics from one or more jobs, and provides metric changes as tuple stream.

Types

  • Notification: The notification type specifies all attributes that can be used to uniquely identify a changed metric value, starting with the domain and instance ids, job and other information, and finally the metric name and its value.
