Threshold document

IBMStreams streamsx.monitoring Toolkit > com.ibm.streamsx.monitoring 1.3.2 > com.ibm.streamsx.monitoring.metrics > MetricsMonitor > Threshold document

The threshold document contains a list of threshold rules. Each threshold rule is defined as a JSON object. This JSON object contains 3 properties:

metricName: String specifying which metric to monitor.

thresholds (optional): JSON object specifying 1 or more thresholds (example format below). There are 4 supported threshold types the user can select from:

  • value: A value to evaluate incoming metric values against.

  • rate: The rate of incoming metric values, given a timeframe.

  • rollingAverage: The rolling average of incoming metric values, given a timeframe.

  • increasePercentage: The percentage that incoming metric values are increasing/decreasing, given a timeframe.

Note that you may choose to add an operator (">", ">=", "<", or "<=") in front of the threshold value to indicate whether you want to monitor for values above or below a certain threshold.

filters (optional): JSON object specifying 1 or more filters. Filters give users the option to further filter which metrics they want monitored. Simply provide an attribute of the incoming metric tuples as a key, and a regular expression as the value (example format below).

The following threshold document monitors the com.ibm.streamsx.monitoring.sample.MetricsSource.MemoryMetrics sample application for when its nResidentMemoryConsumption goes above 10000KB: 

[
  {
    "metricname":"nResidentMemoryConsumption",
    "thresholds":
    {
      "value":">10000",
    },
    "filters":
    {
      "jobName":"com.ibm.streamsx.sample.MetricsSource.MemoryMetrics.*"
    }
]

The following threshold document monitors the com.ibm.streamsx.monitoring.sample.MetricsMonitor.HighCongestion sample application for when its congestionFactor has a rolling average greater than 80 over a timeframe of 20s: 

[
  {
    "metricname":"congestionFactor",
    "thresholds":
    {
      "value":">=80,20000",
    },
    "filters":
    {
      "jobName":"com.ibm.streamsx.sample.MetricsMonitor.HighCongestion.*"
    }
]
