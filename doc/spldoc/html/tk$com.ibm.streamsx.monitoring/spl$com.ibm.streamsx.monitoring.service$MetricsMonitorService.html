Composites

composite MetricsMonitorService

The MetricsMonitorService ingests metric tuples from the MetricsIngestService and monitors the metric values against the threshold rules defined in the application configuration's thresholdDocument property.

Once a threshold is violated, an alert message is sent out, containing the details on the current metric value, the threshold rule that Was triggered. It also includes more details on the metric (from the tuple).
