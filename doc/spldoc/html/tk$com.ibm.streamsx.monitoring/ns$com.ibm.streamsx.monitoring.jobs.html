Operators

  • JobStatusMonitor: This operator uses JobStatusSource and prevents near duplicate notifications being emitted.
  • JobStatusSource: The JobStatusSource operator uses the JMX API to retrieve status updates from one or more jobs, and provides status changes as tuple stream.

Types

  • JobStatusNotification: The JobStatusNotification type specifies attributes that are filled by the JobStatusSource operator
